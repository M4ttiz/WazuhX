# Contributing Guidelines

Thank you for considering contributing to WazuhX! Your help is greatly appreciated. Please read these guidelines to make the process smooth for everyone.

## How to Contribute
- **Fork** the repo and create your branch from `main`.
- **Describe your changes** clearly in pull requests.
- Keep pull requests focused and small where possible.
- Follow established code style and structure.
- Write clear commit messages.

## Code of Conduct
This project follows a [Code of Conduct](CODE_OF_CONDUCT.md). Please review it before contributing.

## Bug Reports and Feature Requests
- Use [GitHub Issues](https://github.com/M4ttiz/WazuhX/issues).
- Search existing issues before opening a new one.

## Pull Requests
1. Fork, branch, and update your code.
2. Ensure code builds and passes tests.
3. Submit a pull request with a clear description.

## Documentation/Typo Fixes
Always welcome!
